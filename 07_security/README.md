# 07 — Spring Security

## 🎯 Learning Objectives
- Secure REST APIs with Spring Security
- Implement JWT-based authentication
- Role-based access control (`@PreAuthorize`)

## 📖 Key Concepts

| Concept | Description |
|---|---|
| **Authentication** | Who are you? (login) |
| **Authorization** | What are you allowed to do? (roles/permissions) |
| `@EnableWebSecurity` | Enable Spring Security config |
| `SecurityFilterChain` | Define which endpoints are protected |
| **JWT** | JSON Web Token — stateless auth token |
