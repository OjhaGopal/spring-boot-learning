# 03 — Spring MVC & REST Controllers

## 🎯 Learning Objectives
- Build REST endpoints with `@RestController`
- Handle HTTP methods: GET, POST, PUT, DELETE
- Use `@PathVariable` and `@RequestParam`
- Return JSON responses

## 📖 Key Annotations

| Annotation | Purpose |
|---|---|
| `@RestController` | Marks class as REST controller (returns data, not views) |
| `@GetMapping` | Handle GET requests |
| `@PostMapping` | Handle POST requests |
| `@PutMapping` | Handle PUT requests |
| `@DeleteMapping` | Handle DELETE requests |
| `@PathVariable` | Extract value from URL path `/users/{id}` |
| `@RequestParam` | Extract query param `/users?name=gopal` |
| `@RequestBody` | Map request JSON body to a Java object |
